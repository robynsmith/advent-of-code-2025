# Advent of Code 2025

My solutions to [Advent of Code 2025](https://adventofcode.com/2025).

## About

Advent of Code is an annual set of Christmas-themed programming puzzles. In 2025, the event runs for **12 days** (December 1-12), with two puzzles released each day.

## Structure

Each day has its own folder containing:
- `challenge.md` - The puzzle description
- `solution.py` - My Python solution
- `input.txt` - My puzzle input (not committed per AoC's request)
